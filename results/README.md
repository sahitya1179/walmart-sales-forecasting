# Results

Generated by `python -m src.main` (or the notebook). These artifacts are
git-ignored — run the pipeline to (re)create them.

- `metrics.csv` — model-comparison table (MSE / RMSE / MAE / MAPE). A reference
  copy holding the paper's reported values ships with the repo and is
  overwritten when you re-run the pipeline.
- `figures/`
  - `mse_vs_model.png`, `rmse_vs_model.png`, `mae_vs_model.png` — Fig. 6/7/8
  - `arima_rf_actual_vs_predicted.png` — Fig. 5
- `tableau/`
  - `model_metrics.csv`, `actual_vs_predicted.csv` — tidy inputs for the
    Tableau dashboards (see `../tableau/README.md`).
